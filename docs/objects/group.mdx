# Group

See the [Group Basics](/docs/basics/groups) guide for an introduction to `Group` in Presenter.js.

The full list of available `Group` properties is below:

| Property                  | Type            | Description                                                        | Default                                         |
| ------------------------- | --------------- | ------------------------------------------------------------------ | ----------------------------------------------- |
| `anchor`                  | `Anchor`        | Defines which point of group sits at `(x, y)`                      | `Anchor.TOP_LEFT`                               |
| `height`                  | `number`        | Height of group, in pixels                                         | `0`                                             |
| `objects`                 | `SlideObject[]` | Objects to include in group                                        | none, provided as first argument to constructor |
| `previewColor`            | `Color \| null` | Debugging tool for previewing the group's position and size        | `null`                                          |
| `rotateOriginPreviewSize` | `number`        | Debugging tool for previewing where group's rotation origin sits   | `0`                                             |
| `rotateOriginX`           | `number`        | Horizontal position of rotation origin in group's coordinate space | `0`                                             |
| `rotateOriginY`           | `number`        | Vertical position of rotation origin in group's coordinate space   | `0`                                             |
| `rotation`                | `number`        | Degrees of rotation                                                | `0`                                             |
| `scale`                   | `number`        | Scale factor to use when rendering group                           | `1`                                             |
| `scaleX`                  | `number`        | Horizontal scale factor to use when rendering group                | `1`                                             |
| `scaleY`                  | `number`        | Vertical scale factor to use when rendering group                  | `1`                                             |
| `skewOriginPreviewSize`   | `number`        | Debugging tool for previewing where group's skew origin sits       | `0`                                             |
| `skewOriginX`             | `number`        | Horizontal position of skew origin in group's coordinate space     | `0`                                             |
| `skewOriginY`             | `number`        | Vertical position of skew origin in group's coordinate space       | `0`                                             |
| `skewX`                   | `number`        | Amount of skew in horizontal direction                             | `0`                                             |
| `skewY`                   | `number`        | Amount of skew in vertical direction                               | `0`                                             |
| `width`                   | `number`        | Width of group, in pixels                                          | `0`                                             |
| `x`                       | `number`        | Horizontal position of group                                       | `0`                                             |
| `y`                       | `number`        | Vertical position of group                                         | `0`                                             |
