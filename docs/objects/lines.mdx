# Lines

Use `Line` for a segment between two points and `Arrow` when that segment needs
an arrowhead.

<LinesDemo />

Both objects use `startX`, `startY`, `endX`, and `endY` for their endpoints,
`width` for stroke thickness, and `color` for the stroke. Animate `drawn`
between `0` and `1` for a drawing effect.

`Line` supports `isRounded` for round end caps and `isDrawnFromCenter` to reveal
outward from its midpoint.

`Arrow` adds:

- `arrowheadSize` to control the head independently of line width
- `isArrowheadFilled` for a solid head
- `isArrowheadDoubled` for heads at both ends
- `isDrawnFromCenter` for a center-out reveal
